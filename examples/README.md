# Example input files

- `spc_sample.csv`
  - Columns: `Sous-groupe`, `Mesure1..Mesure5`
  - Import it and go to the **SPC** page to see Xbar/R charts.

- `msa_sample.csv`
  - Columns: `Piece`, `Operateur`, `Essai`, `Valeur`
  - Import it and go to the **MSA** page. If mapping is not auto-detected, set:
    - Pièce -> `Piece`
    - Opérateur -> `Operateur`
    - Essai -> `Essai`
    - Valeur -> `Valeur`
